#Instalación para compilar sass (opcional)

install node
```
https://nodejs.org/es/
```

install npm
```
npm install
```

install gulp-cli
```
npm install --global gulp-cli
```

In project root folder install gulp gulp-sass browser-sync
```
npm install gulp gulp-sass browser-sync
```

In project folder run gulp watch
```
gulp watch
```
